# Дополнение к README финального проекта

## Возможности

- Сравнение вкладов, накопительных счетов и базовых инвестиционных вариантов.
- Калькуляторы примерного дохода.
- Фильтры и избранное продуктов.
- Анкета и матрица выбора подходящего инструмента.
- Учебные материалы, мини-тесты, прогресс и заметки.
- Админские блоки для контроля источников и состояния API.

## Рекомендуемый запуск

```powershell
cd backend
pip install -r requirements.txt
python app.py
```

```powershell
cd frontend
pip install -r requirements.txt
python app.py
```

Backend: `http://127.0.0.1:5000`

Frontend: `http://127.0.0.1:5001`
