# Fin-gramotnost
Информационный сайт в формате навигации по различным банковским продуктам в сегменте сбережений: вклады, накопительные счета, инвестиционное страхование жизни - с тестовыми вопросами для самопроверки после изучения темы. Сайт должен быть выполнен в формате html для дальнейшего использования.
