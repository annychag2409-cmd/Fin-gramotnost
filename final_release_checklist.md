# Финальный чек-лист релиза 6

## Запуск

- Backend запускается на `http://127.0.0.1:5000`.
- Frontend запускается на `http://127.0.0.1:5001`.
- Страница главной открывается без ошибок.
- В консоли браузера нет критических ошибок.

## Продукты

- Загружаются вклады.
- Загружаются накопительные счета.
- Работает расчет дохода.
- Работают фильтры и избранное.
- История расчетов сохраняется.

## Сравнение

- Таблица сравнения читается.
- Основная анкета показывает результат.
- Матрица сценариев показывает рекомендацию.
- API health panel корректно отражает состояние backend.

## Обучение

- Открывается список уроков.
- Каждый урок открывается отдельно.
- Мини-тест завершается результатом.
- Прогресс сохраняется.
- Заметки сохраняются.
- Резервная копия скачивается.
